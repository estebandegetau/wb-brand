# Wb-brand Extension For Quarto

World Bank Brand Extension for Quarto.

## Installing



```bash
quarto add https://github.com/estebandegetau/wb-brand
```

This will install the extension under the `_extensions` subdirectory.
If you're using version control, you will want to check in this directory.

## Using

This extension installs a [brand.yml](https://posit-dev.github.io/brand-yml/) configuration for World Bank.
